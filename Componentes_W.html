Componentes de WordPress

Los componentes de WordPress

El núcleo de WordPress incluye un conjunto de archivos adicionales que se conocen como temas, plugins y carga de archivos. Cada uno tiene las siguientes funciones:

Temas

Son los archivos que hacen que tu tema funcione. Esto dicta cómo luce tu sitio y de qué manera mostrará tu contenido. WordPress cuenta con una enorme biblioteca de plantillas prediseñadas que puedes modificar y personalizar a tu gusto.

Plugins

Estos archivos son ideales por si necesitas realizar modificaciones o adiciones para la operatividad de tu sitio. Los plugins de WordPress cumplen funciones de apariencia o más técnicas como de SEO, rendimiento, respaldo, seguridad, entre muchas más.

Carga de archivos

Se compone por todos aquellos archivos exteriores que cargues a la plataforma, por ejemplo, videos o imágenes.

Funciones de administración de WordPress

En cuanto a las funciones de administración de WordPress puedes encontrar las siguientes opciones:

Panel de control

Incluye una interfaz donde podrás ver todo lo que hay disponible.

Publicaciones

Aquí puedes agregar y editar las publicaciones que realices.

Medios

Esta función te permite cargar todos los archivos que te mencionamos anteriormente (imágenes, videos). Estos se quedan almacenados en la plataforma para que puedas utilizarlos cuando lo necesites.

Páginas

Sirve para la creación de páginas estáticas para tu sitio web, además de las que ya pueden venir precargadas con el tema que elijas.

Comentarios

Esta función te permite administrar todos los comentarios que los usuarios hagan en alguna de tus publicaciones. Puedes borrar, editar, responder e incluso ocultar lo que consideres necesario.

Apariencia

Aquí podrás administrar todo lo relacionado al tema de WordPress que hayas elegido, por ejemplo, agregar o quitar barras laterales, personalizar los colores de una página, cambiar la tipografía de tu contenido, entre otros aspectos más.

Usuarios

Esta función te permite administrar a las personas que tendrán acceso para editar o solo ver todo lo interno de tu sitio web.

Configuración

Aquí puedes modificar aspectos internos de tu cuenta, por ejemplo, nombre o experiencia de lectura.

Si es la primera vez que tendr ás contacto con las funciones de WordPress, te recomendamos dedicar un buen tiempo para explorar cada opción, ver qué ofrece cada una de ellas y cómo puede ayudarte a crear un sitio web.
